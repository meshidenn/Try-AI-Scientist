- `spec.yaml` template
- `survey/README.md` template
- `results.md` template
- `artifact-audit.md` template
- `paper-review.md` template
- `result-interpretation.md` template
- `next-plan.md` template
- `claim-audit.md` template
- `manifest.json` template
